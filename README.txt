Swedish Snowball Stemmer
================================================================================

Summary
--------------------------------------------------------------------------------

This module implements the Swedish stemming algorithm from the Snowball project
to improve the usability of the core search module.

Read more about the algorithm and stemming in general at:

   http://snowballstem.org/algorithms/swedish/stemmer.html
   http://en.wikipedia.org/wiki/Porter_Stemmer


Installation
--------------------------------------------------------------------------------

1. Copy the swedishsnowball folder to sites/all/modules or to a site-specific
   modules folder.
2. Go to Administer > Site building > Modules and enable the Swedish Snowball Stemmer
   module.


Configuration
--------------------------------------------------------------------------------

This module doesn't offer any configurable options.
